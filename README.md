# Netflix Clone App For Study

Flutter기반 Netflix 디자인 및 일부 기능을 Clone한 App입니다.Cancel changes

## Goal

- Clone Coding을 통한 다양한 Widget Handling 경험
- StatefulWidget vs StatelessWidget 차이
- Widget Lifecycle 학습
- Firebase의 Firestore 연동
